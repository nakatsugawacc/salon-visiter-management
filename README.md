# 来店者管理デモアプリ

QRコード/NFCでの来店者チェックインとリアルタイム管理画面のデモアプリケーションです。

## 🎯 デモの目的

お客様にかんばん形式の来店者管理システムを実演するためのデモアプリです。

## ✨ 主な機能

- **かんばん形式のダッシュボード** - 来店者の現在地を4つのステージで表示
  - 受付
  - 待合室
  - 面談室
  - 完了
- **リアルタイム更新** - ステータス変更が即座に反映
- **経過時間表示** - 各ステージに到着してからの経過時間を表示（10秒ごとに自動更新）
- **ポップアップ通知** - ステータス変更時にリアルタイムで通知表示（5秒で自動消去）
- **QRコードシミュレーション** - 4つのチェックポイント用URLを生成

## 🚀 使い方

### 1. 起動

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` にアクセス

### 2. デモの実演

#### ダッシュボード画面
- メイン画面に4つのチェックポイントが横並びで表示されます
- 各カラムには現在そのステージにいる来店者が表示されます
- 来店者カードには経過時間がリアルタイムで更新されます

#### QRコードシミュレーション
1. 右下の「QRコードを表示」ボタンをクリック
2. 各チェックポイントのURLが表示されます
3. 「テストアクセス →」リンクを別タブで開く
4. 来店者を選択して「チェックイン」ボタンをクリック
5. ダッシュボードでリアルタイムに更新されることを確認
6. 右上にポップアップ通知が表示されます

### 3. チェックポイントURL

- **受付**: `/checkin/reception-001`
- **待合室**: `/checkin/waiting-002`
- **面談室**: `/checkin/meeting-003`
- **完了**: `/checkin/complete-004`

### 4. 固定データ

デモ用に以下の来店者データが用意されています：

- 山田 太郎 (受付)
- 佐藤 花子 (待合室) - 15分経過
- 鈴木 一郎 (待合室) - 8分経過
- 田中 美咲 (面談室) - 25分経過

**リセット機能**: 右上の「リセット」ボタンで初期状態に戻せます

├── src/

│   │   ├── stores/

│   │   │   ├── VisitorCard.svelte         # 来店者カード
│   │   │   ├── Notification.svelte        # 通知ポップアップ
│   │   │   └── QRCodeDisplay.svelte       # QRコード表示
│   │   └── utils.js                  # ユーティリティ関数

## 🛠 技術スタック

- **SvelteKit** - フレームワーク
- **Tailwind CSS** - スタイリング
- **localStorage** - データ永続化（デモ用）
- **Storage Event API** - 別タブ間のリアルタイム同期

## 💡 デモのポイント

1. **リアルタイム性**
   - QRコードをスキャン（別タブでURLアクセス）→ 即座にダッシュボード更新
   - ポップアップ通知で変更を視覚的に表現

2. **直感的なUI**
   - かんばん形式で来店者の流れが一目瞭然
   - カラーコーディングでステージを識別

3. **実用的な情報**
   - 経過時間表示で待ち時間を把握
   - 各ステージの人数を即座に確認

4. **スケーラビリティの提案**
   - 実運用では Supabase や Firebase を使用してサーバー側で管理
   - スタッフへの通知機能追加
   - 統計・分析ダッシュボード
